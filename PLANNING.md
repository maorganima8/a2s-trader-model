## Stack טכני
- Frontend: Next.js 14 + TypeScript + Tailwind CSS (PWA - web app responsive)
- Backend: Next.js API Routes
- DB: PostgreSQL + Prisma
- Auth: NextAuth.js
- WhatsApp: Green API
- Email: Resend
- Push Notifications: Web Push API
- Scheduled Jobs: node-cron
- Integrations: Fireberry Webhook (הצטרפות תלמיד + לייבים), סליקה לחידוש מנוי
- Repo: https://github.com/maorganima8/a2s-trader-model

אם כן לקח:
- תיעוד עסקה ביומן + תמונה/קישור TradingView
- הודעה למנטור עם הפרטים

## יומן מסחר

תיעוד עסקאות - שדות:
- סוג עסקה: לייב / בדיעבד (מופרד בבירור)
- זמן כניסה ויציאה
- קישור לעסקה (TradingView)
- יום בשבוע (אוטומטי)
- נכס (EURUSD, ES וכו')
- קילזון (London Open, NY Open וכו')
- משך עסקה (אוטומטי)
- תהליך (MMXM, PO3 וכו')
- סוג סטאפ
- טווח כניסה (5M, 1H וכו')
- תוצאה: Win / Loss / BE
- R:R
- הערות

אם תוצאה = Win, מוצגות אוטומטית:
- דירוג העסקה 1-10
- כמה אתה גאה בה
- מה היה טוב בעסקה

אם תוצאה = Loss, מוצגת אוטומטית:
- מה הטעות? (נוספת גם לרשימת הטעויות)

תיקיות:
- עסקאות מנצחות - נאספות אוטומטית (Win)
- עסקאות מפסידות - נאספות אוטומטית (Loss)
- סקירה שבועית - התלמיד מעלה כל יום ראשון
- עסקאות בדיעבד - התלמיד מעלה כל יום ראשון

כל יום ראשון - תזכורת אוטומטית לסקירה ועסקאות בדיעבד.
אם לא העלה - התראה למנטור.

רשימת טעויות:
- אוטומטי מכל עסקת Loss
- ידני - התלמיד יכול להוסיף בעצמו

הערות אישיות:
- מקום חופשי לכתוב (יומן אישי)

לוח שנה:
- ירוק = יום רווחי
- אדום = יום מפסיד
- אפור = לא סחר

סטטיסטיקות - אוטומטיות לגמרי:
- Macro: שנים, חודשים
- Results: Win/Loss/BE
- Daily: ימים, קילזונים
- Pair: נכסים
- Setup: סוג סטאפ, טיים פריים, Trade Types
- ביצועים חודשיים: גרפים + מספרים
- לייב ובדיעבד מחושבים בנפרד
