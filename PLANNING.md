# A2S Trader App - תכנון מלא

## על העסק
- שם: Addiction 2 Success (addiction2success.co.il)
- מטרה: להוביל סוחר מתוסכל לסוחר ממומן שמושך כסף תוך 90 ימים
- התחייבות לתוצאה + החזר כספי

---

## Stack טכני
- Frontend: Next.js 14 + TypeScript + Tailwind CSS (PWA - web app responsive)
- Backend: Next.js API Routes
- DB: PostgreSQL + Prisma
- Auth: NextAuth.js
- WhatsApp: Green API
- Email: Resend
- Push Notifications: Web Push API
- Scheduled Jobs: node-cron
- Repo: https://github.com/maorganima8/a2s-trader-model

---

## תפקידים במערכת
- Admin (אביב) - רואה הכל, כל תלמיד, כל מנטור
- Mentor - יוצר פרופיל תלמיד, רואה את התלמידים שלו
- Student - צ'קליסט יומי, דיווח סיום סשן, יומן מסחר

---

## Onboarding - כניסה לתלמיד
- המנטור יוצר פרופיל בשיעור 1 או 2 (בשיתוף מסך עם התלמיד)
- המנטור ממלא את השאלון, התלמיד רואה ומאשר
- בסוף - המערכת שולחת לתלמיד הזמנה (WhatsApp + מייל) עם לינק כניסה
- התלמיד נכנס ומגדיר סיסמה

---

## שאלון בניית פרופיל - שיעור 2
שדות:
1. שם מלא
2. מספר טלפון
3. שוק (פורקס / חוזים עתידיים) - שדה חדש שנוסיף
4. מה מונע ממך לסחור נכון?
5. כמה זמן רוצה להחזיק פוזיציה?
6. כמה עסקאות ביום?
7. מה קורה כשעסקה בגבול?
8. הפחד הכי גדול לפני כניסה?
9. סוג עסקה מחפש?
10. הכרת תהליך (PO3 / MMXM / IRL>ERL)
11. טיים פריים מועדף
12. שעות מסחר מועדפות (עד שעתיים - תלוי בשוק)
13. ימים למסחר
14. אסטרטגיית Pd-Array
15. SMT - כן/לא
16. רמת סיכון ראשון
17. רמת סיכון שלב הבא

הערות:
- המנטור קובע שעות מסחר - המערכת שולחת התראה אוטומטית 30 דקות לפני
- סשן עד שעתיים לפי השוק

## הלולאה היומית

### 30 דקות לפני סשן
- התראה לתלמיד (WhatsApp + מייל + Push)
- תלמיד פותח אפליקציה וממלא צ'קליסט טרום-סשן (תיבות סימון - הרשימה תתווסף בהמשך)

### סיום סשן
- התראה: "סגור את המחשב"
- שאלה: "לקחת עסקה?"

אם לא לקח:
- שאלה: "מה הסיבה?"
- שליחת הודעה למנטור/אביב ב-WhatsApp עם שם התלמיד והסיבה

אם כן לקח:
- תיעוד עסקה ביומן + העלאת תמונה/קישור TradingView
- שליחת הודעה לאביב עם הפרטים

---

## יומן מסחר

### תיעוד עסקאות - שדות
- סוג עסקה: לייב / בדיעבד (מופרד בבירור)
- זמן כניסה ויציאה
- קישור לעסקה (TradingView)
- יום בשבוע (אוטומטי)
- נכס (EURUSD, ES וכו')
- קילזון (London Open, NY Open וכו')
- משך עסקה (אוטומטי)
- תהליך (MMXM, PO3 וכו')
- סוג סטאפ
- טווח כניסה (5M, 1H וכו')
- תוצאה: Win / Loss / BE
- R:R
- הערות

אם תוצאה = Win, מוצגות אוטומטית:
- דירוג העסקה 1-10
- כמה אתה גאה בה
- מה היה טוב בעסקה

אם תוצאה = Loss, מוצגת אוטומטית:
- מה הטעות?

### תיקיות
- עסקאות מנצחות - נאספות אוטומטית (Win)
- עסקאות מפסידות - נאספות אוטומטית (Loss)
- סקירה שבועית - התלמיד מעלה כל יום ראשון
- עסקאות בדיעבד - התלמיד מעלה כל יום ראשון

כל יום ראשון - תזכורת אוטומטית לתלמיד להעלות סקירה ועסקאות בדיעבד.
אם לא העלה - התראה לאביב.

### רשימת טעויות
- אוטומטי מכל עסקת Loss
- ידני - התלמיד יכול להוסיף בעצמו

### הערות אישיות
- מקום חופשי לכתוב (כמו יומן אישי)

### לוח שנה
- כל יום מסחר מוצבע אוטומטית: ירוק (רווח) / אדום (הפסד) / אפור (לא סחר)

### סטטיסטיקות - אוטומטיות לגמרי
Macro: שנים, חודשים
Results: Win/Loss/BE
Daily: ימים, קילזונים
Pair: נכסים
Setup: סוג סטאפ, טיים פריים, Trade Types
ביצועים חודשיים: גרפים + מספרים
